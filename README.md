=========================================
PERSONAL PORTFOLIO WEBSITE
=========================================

PROJECT TITLE:
Personal Portfolio 

DESCRIPTION:
This is a modern and responsive Personal Portfolio
Website developed using HTML and CSS. The website
showcases personal information, skills, education,
projects, achievements, and contact details in a
professional and attractive design.

-----------------------------------------
FEATURES
-----------------------------------------

✓ Modern User Interface
✓ Fully Responsive Design
✓ Animated Gradient Background
✓ Glassmorphism Effect
✓ Hero Section
✓ About Me Section
✓ Skills Section
✓ Education Section
✓ Projects Showcase
✓ Achievements Section
✓ Contact Information
✓ Social Media Links
✓ Smooth Scrolling
✓ Mobile Friendly Layout

-----------------------------------------
TECHNOLOGIES USED
-----------------------------------------

2. CSS3

-----------------------------------------
HOW TO RUN
-----------------------------------------

1. Download all project files.
2. Place index.html and style.css
   in the same folder.
3. Open index.html in a web browser.

OR

1. Open project folder in VS Code.
2. Install Live Server extension.
3. Right click on index.html.
4. Select "Open with Live Server".
